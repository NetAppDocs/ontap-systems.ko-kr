= DS212C, DS224C 또는 DS460C 쉘프 콜드 교체
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
운영 중인 디스크가 있는 프로덕션 시스템에서 드라이브 쉘프를 교체할 때는 콜드 쉘프 교체를 수행해야 합니다. 이 작업은 HA 쌍의 컨트롤러를 중지해야 하는 중단 작업입니다.

이 절차는 IOM12, IOM12B 또는 IOM12C 모듈이 있는 쉘프에 적용됩니다.

NetApp 기술 문서 사용 https://kb.netapp.com/onprem/ontap/hardware/How_to_replace_a_shelf_chassis_using_a_cold_shelf_removal_procedure["콜드 쉘프 제거 절차를 사용하여 쉘프 섀시를 교체하는 방법"].
