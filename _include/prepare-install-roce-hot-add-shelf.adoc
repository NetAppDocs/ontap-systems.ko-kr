시스템에서 RoCE 지원 PCIe 카드 또는 I/O 모듈 사용을 지원하는 경우, HA 쌍에는 핫 추가하려는 쉘프 수를 지원할 수 있는 충분한 RoCE 지원 이더넷 포트가 있어야 합니다.

.단계
. 핫 추가할 각 쉘프에 대해 각 컨트롤러에 RoCE 지원 포트가 2개 있는지 확인합니다.
+
이러한 포트는 시스템에서 지원하는 대로 컨트롤러 온보드, RoCE 지원 PCIe 카드, 둘의 조합 또는 RoCE 지원 I/O 모듈에 있을 수 있습니다.

. HA 쌍에 사용 가능한 RoCE 지원 포트가 충분하지 않은 경우, 시스템에서 지원하는 올바른 컨트롤러 슬롯에 추가 PCIe 카드 또는 I/O 모듈을 설치하십시오.
+
.. 시스템에 맞는 컨트롤러 슬롯을 확인하십시오.  https://hwu.netapp.com["NetApp Hardware Universe를 참조하십시오"^]를 참조하십시오.
.. PCIe 카드 또는 I/O 모듈 설치 지침은 시스템 설명서를 참조하십시오.
