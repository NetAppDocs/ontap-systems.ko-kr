컨트롤러 모듈을 교체할 때 손상된 컨트롤러 모듈에서 교체 컨트롤러 모듈로 캐싱 모듈을 이동해야 합니다.


NOTE: Ver2 컨트롤러 모듈에는 FAS8300에는 캐싱 모듈 소켓이 하나만 있습니다. FAS8700에는 VER2 컨트롤러 모듈이 없습니다. 캐싱 모듈 기능은 소켓 제거의 영향을 받지 않습니다.

다음 애니메이션, 그림 또는 기록된 단계를 사용하여 캐싱 모듈을 새 컨트롤러 모듈로 이동할 수 있습니다.

.애니메이션 - 캐싱 모듈을 이동합니다

.단계
. 아직 접지되지 않은 경우 올바르게 접지하십시오.
. 캐시 모듈을 손상된 컨트롤러 모듈에서 교체 컨트롤러 모듈로 이동합니다.
+
.. 캐싱 모듈 끝에 있는 파란색 분리 탭을 누르고 모듈을 위로 돌린 다음 소켓에서 모듈을 분리합니다.
.. 캐시 모듈을 교체 컨트롤러 모듈의 동일한 소켓에 끼웁니다.
.. 캐싱 모듈의 가장자리를 소켓에 맞추고 모듈을 소켓에 완전히 끼웁니다.
.. 캐싱 모듈을 아래로 돌려 마더보드쪽으로 이동합니다.
.. 파란색 버튼을 사용하여 캐싱 모듈의 끝에 손가락을 놓고 캐싱 모듈 끝을 단단히 누른 다음 잠금 버튼을 들어 올려 캐싱 모듈을 제자리에 고정합니다.
