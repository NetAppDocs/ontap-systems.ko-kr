= AFF C80의 주요 사양
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
다음은 단일 고가용성 쌍의 AFF C80 스토리지 시스템에 대한 선택 사양입니다.  이 스토리지 시스템의 전체 사양을 보려면 NetApp Hardware Universe (HWU)를 방문하세요.



== AFF C80의 주요 사양

* 플랫폼 구성: AFF C80 단일 섀시 HA 쌍
* 최대 원시 용량: 14.7360 PB
* 메모리: 512.0000GB
* 폼 팩터: 2개의 HA 컨트롤러와 48개의 드라이브 슬롯이 있는 4U 섀시
* ONTAP 버전: ONTAP: 9.16.1P2
* PCIe 확장 슬롯: 18개
* 최소 ONTAP 버전: ONTAP 9.16.1RC1 == 확장 최대값
* 유형: NAS; HA 쌍: 12; 원시 용량: 176.8PB / 157.0PiB; 최대 메모리: 6144GB
* 유형: SAN; HA 쌍: 6; 원시 용량: 88.4PB / 78.5PiB; 최대 메모리: 3072GB
* 유형: HA 쌍; 원시 용량: 14.7PB / 13.1PiB; 최대 메모리: 512.0000
