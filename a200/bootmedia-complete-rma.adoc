= 장애가 발생한 부품을 NetApp-AFF A200으로 반환합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
키트에 동봉된 RMA 지침에 따라 부팅 미디어를 교체한 후 고장난 부품을 NetApp 으로 반송하십시오. 보세요 https://mysupport.netapp.com/site/info/rma["부품 반환 및 교체"] 자세한 내용은 해당 페이지를 참조하십시오. AFF A200 시스템은 수동 부팅 미디어 복구 절차만 지원합니다. 자동 부팅 미디어 복구는 지원되지 않습니다.
