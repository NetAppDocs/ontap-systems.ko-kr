= 장애가 발생한 부품을 NetApp-AFF A300에 반환
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
부팅 미디어를 교체한 후 고장난 부품을 NetApp 으로 반송하십시오. AFF A300 시스템은 수동 부팅 미디어 복구 절차만 지원합니다. 자동 부팅 미디어 복구는 지원되지 않습니다.

키트와 함께 제공된 RMA 지침에 설명된 대로 오류가 발생한 부품을 NetApp에 반환합니다.  https://mysupport.netapp.com/site/info/rma["부품 반환 및 교체"]자세한 내용은 페이지를 참조하십시오.
