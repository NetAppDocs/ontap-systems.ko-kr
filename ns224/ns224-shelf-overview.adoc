= 개요 - NS224 쉘프
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp NS224 드라이브 쉘프는 최대 24개의 드라이브를 장착할 수 있는 2U 랙 유닛입니다. NS224 드라이브 쉘프는 2.5" 소형 폼 팩터 드라이브와 호환되며 SAS 및 SATA 인터페이스를 모두 지원합니다.
